# Hero background

Drop your custom hero image here as:

    hero-background.jpg

Supported file extensions: `.jpg`, `.jpeg`, `.png`, `.webp` (filename stem
must be exactly `hero-background`).

The site will pick the file up automatically on the next build / dev reload —
no code change required. If no matching file is found, the hero falls back to
the animated `<GurdwaraBg />` SVG background.

## Recommended specs

- **Aspect ratio:** wide landscape (16:9 or 21:9 works best — hero is full-viewport)
- **Resolution:** 2400 × 1350 px (or larger for 4K displays)
- **File size:** ≤ 500 KB after compression — run through tinypng.com / squoosh.app
- **Subject framing:** keep critical content (shrine, dome, etc.) in the upper
  two-thirds; the bottom is darkened by the overlay and the navbar covers the
  top 72 px on small screens
- **Tone:** the hero overlay darkens the image (~70% → ~55% → ~80% top to
  bottom). Choose a bright, well-lit photograph or it will look muddy after
  the overlay.
